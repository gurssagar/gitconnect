# Contributing to GitConnect

Thank you for your interest in contributing to GitConnect! We welcome contributions from the community.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/gitconnect.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

### Setup

```bash
npm install
npm run build
```

### Running Tests

```bash
npm test
npm run test:coverage
```

### Linting

```bash
npm run lint
npm run lint:fix
npm run format
```

## Making Changes

1. Make your changes in the `src/` directory
2. Add tests for new functionality
3. Ensure all tests pass: `npm test`
4. Ensure code is linted: `npm run lint`
5. Update documentation if needed

## Commit Messages

Please use clear and meaningful commit messages:

- `feat: add new command`
- `fix: resolve issue with X`
- `docs: update README`
- `test: add tests for Y`
- `refactor: improve code structure`

## Pull Request Process

1. Update the README.md if needed
2. Ensure all tests pass
3. Update version numbers if applicable
4. Submit PR with clear description

## Code of Conduct

- Be respectful
- Welcome newcomers
- Focus on constructive feedback
- Respect different viewpoints

## Questions?

Feel free to open an issue for discussion!
